# Keio Easylist

慶應義塾大学の学生のための授業情報を集めているサイトです。

## Features

- 日吉・三田・矢上キャンパスで開講されている全ての学部の履修の組み方
- 一般教養の口コミ情報
- About Us

## Tech Stack

**Client:** React, Next.js, TypeScript, TailwindCSS

**Server:** Next.js (API Routes)

**DataBase:** PostgreSQL

**ORM:** Prisma
